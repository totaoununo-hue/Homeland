# Homeland

Este projeto é um site estático para uma pagina sobre a terra natal, desenvolvido utilizando conhecimentos em HTML e CSS.

## Conhecimentos Utilizados

- HTML5 para estruturação do conteúdo.
- CSS3 para estilização, incluindo responsividade com media queries.
- Font-face para aplicação de fontes personalizadas.
- Estruturação modular de CSS em blocos (header, body, footer, etc.).
- Normalização de estilos com normalize.css.

## Como Foi Desenvolvido

- Estrutura do projeto organizada em pastas: blocks/, images/, pages/, vendor/.
- Desenvolvimento no VS Code, com foco em design responsivo.
- Controle de versão com Git.
- Hospedagem no GitHub Pages.
